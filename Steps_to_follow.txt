1) Create new env in the folder.
2) Activate the env '.\env_name\Scripts\activate'
3) pip freeze > backend/requirements.txt
4) run: npm install
5) cd my-app
6) run: npm run dev
7) Open new terminal and run: '.\env_name\Scripts\activate'
8) run: cd my-app/backend
9) py manage.py makemigrations
10) py manage.py migrate
11) py manage.py createsuperuser (This is the Admin)
12) py manage.py shell 

13) Run the following lines:

	from django.contrib.auth import get_user_model
	User = get_user_model()

	# Replace 'your_username' with the username you just created
	u = User.objects.get(username='your_username')

	# Remove staff access
	u.is_staff = False
	u.save()

	# Exit the shell
	exit()

14) run: py manage.py loaddata initial_data.json
15) Create Owner,and Employee using Django Admin.(NOTE: Owner should be just the 'Superuser' and Employee should be just the 'Staff')
16) Create Retailers/Dealers/Builders Users using the Admin Django method or using the Owner profile.
